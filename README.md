# Руководство по запуску лабораторных

## Параметры по умолчанию

Если запускать программу без аргументов, используются:

- Размеры изображений: `1024x768,1280x960,2048x1536`
- Количество прогонов на размер: `3`
- Потоки CPU: `2,4,6,8,10,12,14,16`
- Папка входных изображений: `test_images`
- Папка выходных изображений: `output_images`
- Пайплайн по умолчанию: `invert-contrast` (ЛБ2)

---

## ЛБ1 (старое задание): Сдвиг + Цвет границы + Размытие

### Кратко по заданию

- Загрузить цветное изображение.
- Выполнить сдвиг пикселей по X и Y.
- Пограничные области заполнить цветом `RGB(187, 38, 73)`.
- Выполнить свертку с ядром размытия по каждому каналу RGB.
- Сохранить результат.
- Провести тесты на размерах:
  - `1024 x 768`
  - `1280 x 960`
  - `2048 x 1536`
- Для каждого случая сделать 3 прогона и посчитать среднее время.
- Оценить производительность для потоков:
  - `2, 4, 6, 8, 10, 12, 14, 16`

### Команда запуска (ЛБ1)

```powershell
.\gradlew run --args="--pipeline shift-blur"
```

Явно со всеми параметрами:

```powershell
.\gradlew run --args="--pipeline shift-blur --shiftX 50 --shiftY 30 --borderColor 187,38,73 --iterations 3 --sizes 1024x768,1280x960,2048x1536 --threads 2,4,6,8,10,12,14,16"
```

## ЛБ2 (новое задание): Инверсия + Контрастная свертка

### Кратко по заданию

- Загрузить цветное изображение.
- Выполнить инверсию каждого канала: `new = 255 - old`.
- Выполнить контрастную свертку по каждому каналу RGB.
- Сохранить результат.
- Провести тесты на размерах:
  - `1024 x 768`
  - `1280 x 960`
  - `2048 x 1536`
- Для каждого случая сделать 3 прогона и посчитать среднее время.
- Оценить производительность для потоков:
  - `2, 4, 6, 8, 10, 12, 14, 16`

### Команда запуска (ЛБ2)

Запуск по умолчанию уже использует ЛБ2:

```powershell
.\gradlew run
```

Явно для ЛБ2:

```powershell
.\gradlew run --args="--pipeline invert-contrast"
```

## Входные изображения

- Программа ищет изображения в папке `test_images`.
- Ожидаемые имена файлов: `test_<width>x<height>.png`, например:
  - `test_1024x768.png`
  - `test_1280x960.png`
  - `test_2048x1536.png`
- Если файлов нет, программа создаст тестовые изображения автоматически.
